# Doctors Portal 

live link: https://doctors-portal-dfd36.web.app/

## This Project consists of:

- HTML
- CSS
- Javascript
- React
  - react component
  - react hooks
  - useState
  - useEffect
  - custom hook
  - react router(v-6)
  - custom api
- Material UI
- Firebase Authentication
  - google sign in
  - email and password sign in
  - jwt token enabled
- Mongodb to handle data
- Image upload in mongodb
- Heroku for server deploment
